CSS Grid 布局

CSS Grid 是二维布局系统，可以同时处理行和列。它是 Flexbox 的补充，适合复杂的网格布局。

基本概念

  • Grid 容器: 使用 display: grid 创建网格容器
  • Grid 项目: 容器内的直接子元素
  • 网格线: 网格的分割线
  • 网格轨道: 两条网格线之间的空间

容器属性

.grid-container {


  display: grid;


  grid-template-columns: repeat(3, 1fr);


  grid-template-rows: auto;


  gap: 20px;


}

项目属性

  • grid-column: 指定项目占据的列
  • grid-row: 指定项目占据的行
  • grid-area: 指定项目占据的区域

适用场景

  • 复杂的二维布局
  • 响应式网格系统
  • 卡片网格布局
  • 页面整体布局
