ESLint 代码检查

ESLint 是一个可插拔的 JavaScript 代码检查工具，用于发现和修复代码问题。

配置文件

{


  "extends": ["eslint:recommended"],


  "rules": {


    "no-console": "warn",


    "no-unused-vars": "error"


  }


}

常用规则

  • no-console: 禁止使用 console
  • no-unused-vars: 禁止未使用的变量
  • eqeqeq: 要求使用 === 和 !==
  • semi: 要求或禁止分号

与 Prettier 集成

{


  "extends": [


    "eslint:recommended",


    "plugin:prettier/recommended"


  ]


}

优势

  • 统一代码风格
  • 提前发现错误
  • 提高代码质量
  • 团队协作规范
