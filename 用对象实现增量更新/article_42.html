Vue 组件通信

Vue 组件间有多种通信方式，适用于不同的场景。

Props 和 Events

<!-- 父组件 -->


<ChildComponent :data="parentData" @update="handleUpdate" />
<!-- 子组件 -->


<template>


  <div @click="$emit('update', newData)">


    {{ data }}


  </div>


</template>


<script>


export default {


  props: ['data'],


  emits: ['update']


}


</script>

provide/inject

// 祖先组件


provide() {


  return {


    theme: this.theme


  }


}
// 后代组件


inject: ['theme']

Vuex/Pinia

  • 全局状态管理
  • 跨组件共享数据

事件总线（已废弃）

  • 使用全局事件总线
  • Vue 3 推荐使用 mitt 库

最佳实践

  • 父子组件用 props/events
  • 跨层级用 provide/inject
  • 全局状态用状态管理
