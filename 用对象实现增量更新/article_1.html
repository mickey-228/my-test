React Hooks 基础(我被修改了）

React Hooks

Hooks 是 React 16.8 引入的新特性，允许在函数组件中使用状态和生命周期。它解决了类组件中代码复用困难、逻辑分散等问题。

常用 Hooks

  • useState: 管理组件状态，返回状态值和更新函数
  • useEffect: 处理副作用，如数据获取、订阅、DOM 操作等
  • useContext: 使用上下文，避免 props 层层传递
  • useMemo: 性能优化，缓存计算结果
  • useCallback: 缓存函数引用，避免子组件不必要的重渲染
  • useRef: 获取 DOM 引用或保存可变值

使用示例

// useState 示例


const [count, setCount] = useState(0);


const increment = () => setCount(count + 1);
// useEffect 示例


useEffect(() => {


  document.title = Count: ${count};


  return () => {


    // 清理函数


  };


}, [count]); // 依赖数组

Hooks 规则

  • 只能在函数组件顶层调用
  • 不能在条件语句、循环中调用
  • 自定义 Hooks 必须以 use 开头
