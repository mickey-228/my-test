Vue 响应式原理

Vue 响应式原理

Vue 的响应式系统是其核心特性之一，能够自动追踪数据变化并更新视图。Vue 2.x 使用 Object.defineProperty 实现响应式，Vue 3.x 使用 Proxy API。

核心概念

  • 响应式数据: 当数据发生变化时，自动更新依赖该数据的视图
  • 依赖收集: 在渲染过程中追踪哪些数据被使用，建立依赖关系
  • 派发更新: 当数据变化时，通知所有依赖该数据的组件进行更新
  • 观察者模式: 使用观察者模式实现数据与视图的自动同步

Vue 2.x 实现方式

Object.defineProperty(obj, 'key', {


  get() {


    // 依赖收集


    return value;


  },


  set(newVal) {


    // 派发更新


    value = newVal;


  }


});

Vue 3.x Proxy 实现

const reactive = new Proxy(obj, {


  get(target, key) {


    // 依赖收集


    return target[key];


  },


  set(target, key, value) {


    // 派发更新


    target[key] = value;


    return true;


  }


});

Proxy 优势

  • 可以监听数组变化（Vue 2 需要特殊处理）
  • 可以监听对象属性的添加和删除
  • 性能更好，不需要递归遍历所有属性
  • 支持 Map、Set 等更多数据结构
  • 代码更简洁，不需要为每个属性单独定义

响应式 API

  • reactive(): 创建响应式对象
  • ref(): 创建响应式基本类型值
  • computed(): 计算属性
  • watch(): 监听数据变化
