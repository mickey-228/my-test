TypeScript 类型系统

TypeScript 是 JavaScript 的超集，添加了静态类型检查。它可以在编译时发现错误，提供更好的 IDE 支持，并提高代码的可维护性。

基础类型

  • string: 字符串类型
  • number: 数字类型（包括整数和浮点数）
  • boolean: 布尔值类型
  • null 和 undefined: 空值类型
  • void: 无返回值
  • any: 任意类型（应避免使用）
  • unknown: 类型安全的 any

数组和对象

let arr: number[] = [1, 2, 3];


let obj: { name: string; age: number } = { name: 'Tom', age: 20 };

高级类型

  • 联合类型: string | number - 可以是多种类型之一
  • 交叉类型: A & B - 同时满足多个类型
  • 接口: interface - 定义对象结构
  • 类型别名: type - 为类型创建别名
  • 泛型: - 创建可重用的类型
  • 字面量类型: 'success' | 'error' - 精确的字符串值

接口示例

interface User {


  name: string;


  age: number;


  email?: string; // 可选属性


}
function getUser(id: number): User {


  // ...


}

泛型示例

function identity<T>(arg: T): T {


  return arg;


}
let output = identity<string>('hello');

优势

  • 早期发现错误
  • 更好的代码提示
  • 重构更安全
  • 自文档化代码
